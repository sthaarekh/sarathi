# Sarathi ‚Äì Your Ultimate Club Management System

## About Sarathi

Sarathi is a centralized platform that simplifies the process of discovering, exploring, and managing university clubs. Whether you're a student looking for the perfect club, a club manager handling events, or university administration overseeing activities, **Sarathi** makes everything seamless and efficient. 

### Key Features:
- **All-in-One Club Platform** ‚Äì Manage all club activities in a single system.
- **Discover & Connect** ‚Äì Explore clubs based on interests, activities, and events.
- **Seamless Club Management** ‚Äì Effortless organization for club managers and university admins.
- **Centralized & User-Friendly** ‚Äì No more scattered club details‚Äîeverything in one place!

## Tech Stack

- **MongoDB** ‚Äì NoSQL database for efficient data storage
- **Express.js** ‚Äì Backend framework for smooth API handling
